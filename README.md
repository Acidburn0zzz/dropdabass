Drop da Bomb
------------

Protocol:

Server: "HELLO\n"
Server: "MAP\n"
Server: "WIDTHxHEIGHT\n"
Server: "Map\n"
Server: Map
    + = indestructible wall
    . = walkable grass
    # = destructible rock
Server: "STARTPOSITION\n"
Server: X,Y\n

Game starts:
Server: "PLAYERS\n"
Server: ID X,Y

When ready to accept new command:
Server: "OK\n"
